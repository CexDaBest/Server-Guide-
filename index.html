Comandos útiles

/scoreboard

Usa este comando para **crear objetivos** (vidas, puntos, rondas), **modificar valores** y **mostrar** tablas en pantalla. Contiene subcomandos como: objectives add (crear objetivo), players add/set/test (gestionar valores por jugador) y objectives setdisplay (mostrar en sidebar, list, belowName). Útil para triggers condicionales con `/execute` y para almacenar resultados entre rondas.

/function

Ejecuta archivos `.mcfunction` que contienen listas de comandos. Ideal para **secuencias repetitivas**: iniciar ronda, dar kits, limpiar arena, teleports y mensajes. Organiza funciones por carpetas (p. ej. `minigames/start.mcfunction`) y llámalas desde command blocks o al iniciar el servidor.

/setblock

Coloca o reemplaza un bloque en coordenadas exactas. Sirve para **activar trampas**, colocar botones/palancas o crear puntos de spawn temporales. Soporta estados de bloque y datos NBT en algunos casos.

/fill

Rellena un volumen con un bloque o lo limpia (air). Perfecto para **restaurar arenas** entre partidas o crear muros y plataformas rápidamente. Opciones: `replace`, `destroy`, `keep` para controlar comportamiento sobre bloques existentes.

/clone

Copia una región y la pega en otra ubicación. Úsalo para replicar niveles, obstáculos o spawn points idénticos sin reconstruirlos.

/teleport ( /tp )

Teletransporta jugadores/entidades a coordenadas o a otros jugadores. Soporta orientación (yaw/pitch) para que los jugadores miren hacia un objetivo al aparecer. Muy útil para puntos de inicio, zonas seguras o teleports de emergencia.

/give

Entrega ítems o kits con NBT opcional (encantamientos, nombres, atributos). Úsalo para dar armas, herramientas o ítems especiales al comenzar cada ronda.

/gamemode

Cambia el modo de juego (survival, creative, adventure, spectator). Útil para fases: preparación (creative), juego (survival/adventure), observador (spectator).

/effect

Aplica efectos de poción (duración, nivel). Úsalo para power-ups (velocidad, fuerza) o penalizaciones (slowness, blindness).

/title

Muestra títulos y subtítulos grandes en pantalla; perfecto para anunciar inicio/fin de rondas, ganador o instrucciones rápidas.

/playsound

Reproduce sonidos a jugadores concretos. Categorías: ambient, block, entity, music, weather, ui, player. **Sonidos comunes:** minecraft:entity.player.levelup, minecraft:entity.experience_orb.pickup, minecraft:block.note_block.pling, minecraft:entity.ender_dragon.growl, minecraft:music.game, minecraft:ambient.cave. Usa volumen y pitch para variar la señalización.

/summon

Genera mobs u objetos con NBT; útil para jefes, objetivos o NPCs que den misiones.

/execute

Comando avanzado para condicionales y lógica: ejecutar acciones **solo si** se cumple una condición (p. ej. puntuación en scoreboard, presencia en área). Es la base para reglas complejas de minijuegos.

Consejos rápidos

Prueba todo en un mundo de pruebas; organiza tus `.mcfunction`; usa /gamerule doDaylightCycle false para controlar el tiempo.
